# 🐍 Python Basics Projects

This repository contains beginner-friendly Python projects created while learning core programming concepts.

The goal of this repo is to practice and demonstrate:
- Python syntax
- Problem solving
- Functions and loops
- File handling
- Error handling

---

## 📂 Projects Included

### 1. Calculator
A simple command-line calculator that performs:
- Addition
- Subtraction
- Multiplication
- Division

Run:
```bash
python calculator.py
```

---

### 2. Number Guessing Game
A game where the user tries to guess a randomly generated number.

Features:
- Hints (Too high / Too low)
- Attempt counter

Run:
```bash
python guessing_game.py
```

---

### 3. To-Do List (CLI)
A command-line task manager with file saving.

Features:
- Add tasks
- Delete tasks
- Persistent storage using a text file

Run:
```bash
python todo_list_cli.py
```

---

### 4. Password Generator
Generates secure random passwords.

Features:
- Custom password length
- Uses letters, numbers, and symbols

Run:
```bash
python password_generator.py
```
